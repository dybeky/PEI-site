# COBRA PEI #1 - Формы для сервера Unturned

Простая статичная страница с формами для отправки идей и жалоб на сервере Unturned.

## Как использовать

Просто откройте `index.html` в браузере или разместите на GitHub Pages.

## Развертывание на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы
3. Перейдите в Settings → Pages
4. Выберите ветку main и папку root
5. Сохраните и ваш сайт будет доступен по адресу: `https://ваш-username.github.io/название-репозитория/`
